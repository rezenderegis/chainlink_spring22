# BlockReceipts Smart Contract

## Introduction 

Welcome!

The file of NFTReceipts.sol could be deployed to the goerli test network, since that network is required to be used for alchemy, and then use the script to mint NFT tokens.

## Deploy smart contract to goerli test network

Please try to use Remix IDE, and open your metamask to select a goerli test network, so that when you are in Remix select the Injected Web3 environment, select the smart contract NFTReceipts.sol, and hit on the section of Deploy.
